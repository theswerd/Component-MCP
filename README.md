# Component MCP

This is an experimental MCP App built for creating and sharing minimal component previews. Your AI can generate a React component with notches for you to see variations of it directly in your chat. 

It has integrating sharing features so you can send to your teammates, or add directly to your project via a generated ShadCN registry.

## Built with

- MCP App SDK
- Bun
- React + Vite
- [Freestyle VMs](https://www.freestyle.sh/products/vms) for hosting the component previews
- [Freestyle Git](https://www.freestyle.sh/products/git) for hosting the component code
- [Freestyle Deploys](https://www.freestyle.sh/products/deploys) for hosting main app server and component preview servers
